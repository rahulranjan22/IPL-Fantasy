# 🏏 CricketDream — IPL Fantasy Cricket

> Full-stack IPL Fantasy platform. Pick your Dream XI, score points, win prizes.

## Stack
| Layer    | Tech                  | Host     |
|----------|-----------------------|----------|
| Frontend | Next.js 14 TypeScript | Vercel   |
| Backend  | Flask (Python) API    | Render   |
| Database | PostgreSQL            | Supabase |
| Auth     | Supabase Auth + JWT   | Supabase |

## Pages
| URL | Page |
|-----|------|
| `/` | Homepage |
| `/matches` | All IPL fixtures |
| `/matches/[id]` | Match detail + scorecard |
| `/players` | Player database |
| `/players/[id]` | Player profile |
| `/teams/build` | Dream XI builder |
| `/my-teams` | Your fantasy teams |
| `/leaderboard` | Season rankings |
| `/profile` | User profile |
| `/admin` | Admin panel |
